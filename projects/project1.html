Programa de fidelidade com clusterização de clientes Objetivo Usei Python, estatística e técnicas não-supervisinadas de Machine Learning para Segmentar um grupo de clientes com base em suas características para performance de compra, a fim de selecionar grupos de clientes para formar um programa de Fidelidade com o objetivo de aumentar a receita da empresa. Resultado O resultado desta solução, caso fosse implementada, seria de R$15MM de dólares de receita anual de acordo com a métrica XYZ. Ferramentas utilizadas Python Estatística Unsupervised Machine Learning
